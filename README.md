# Proyecto servicio para veterinaria

El objetivo del proyecto es

- Crear una plataforma que permita la gestión de citas para una veterinaria.

Requisitos funcionales (features)

- El cliente debe poder registrarse en la plataforma.
- El cliente puede registrar a su mascota una vez se registre.
- El dueño puede ver el registro del cliente.

Para clonar el repositorio:

Si no tiene configurada una clave SSH

```bash
git clone https://github.com/juanchopi37/pets-service.git
```

Si ya la tiene configurada, puede usar el siguiente comando:

```bash
git clone git@github.com:juanchopi37/pets-service.git
```

Entrar en el proyecto

```bash
cd pets-service
```

Instalar dependencias

```bash
npm install
```

Abrir Visual Studio Code

```bash
code .
```

Correr proyecto

```bash
npm run dev
```
